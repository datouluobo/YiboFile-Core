## 概述

OoiMRR 提供了强大的文件预览功能，支持多种常见文件格式的即时预览。本文档详细列出了所有支持的格式、实现状态以及未来计划。

### 预览架构

```
PreviewFactory (工厂)
    ↓
FileTypeManager (类型识别)
    ↓
IPreviewProvider (预览提供者接口)
    ↓
具体预览实现 (ImagePreview, TextPreview, etc.)
```

### 统计摘要

| 类别 | 支持格式数 | 完全支持 | 部分支持 | 计划支持 |
|------|-----------|---------|---------|---------|
| 图片 | 16 | 13 | 2 | 1 |
| 文本/代码 | 62+ | 62+ | 0 | 0 |
| 视频 | 17 | 11 | 0 | 6 |
| 音频 | 17 | 17 | 0 | 0 |
| 文档 | 14 | 7 | 0 | 7 |
| 压缩包 | 16 | 3 | 0 | 13 |
| 字体 | 2 | 0 | 0 | 2 |
| 其他 | 5 | 2 | 0 | 3 |
| **总计** | **149+** | **115+** | **2** | **32+** |

## 技术限制

### 当前限制

1. **DOC格式**
   - 依赖 Microsoft Word COM 组件
   - 未安装Word时预览受限
   - 建议转换为DOCX

2. **旧版Office格式（XLS, PPT）**
   - 需要Office Interop或第三方库
   - 转换为新格式效果更好

3. **专有格式**
   - PSD: 仅显示合并图层
   - AI: 不支持
   - Pages/Numbers/Keynote: 需要macOS环境

4. **视频编解码器**
   - 取决于系统安装的编解码器
   - 推荐安装 K-Lite Codec Pack

5. **CAD文件**
   - DWG依赖第三方库
   - 部分高级特性不支持

### 性能考虑

| 文件类型 | 大小限制 | 建议 |
|---------|---------|------|
| 图片 | < 50MB | 更大图片使用优化加载 |
| 文本 | < 10MB | 超大文本建议外部编辑器 |
| 视频 | < 2GB | 大视频可能加载较慢 |
| 音频 | < 500MB | 无特殊限制 |
| 文档 | < 100MB | 大文档转换可能耗时 |
| 压缩包 | < 1GB | 大压缩包列表加载较慢 |

2. 在 `Previews/` 目录创建新的预览类
3. 实现 `IPreviewProvider` 接口
4. 在 `PreviewFactory.cs` 中添加映射
5. 更新本文档

### 预览器模板

```csharp
using System;
using System.Windows;
using System.Windows.Controls;

namespace OoiMRR.Previews
{
    public class MyFormatPreview : IPreviewProvider
    {
        public UIElement CreatePreview(string filePath)
        {
            try
            {
                // 创建主容器
                var mainPanel = new DockPanel();
                
                // 创建标题栏
                var titlePanel = PreviewHelper.CreateTitlePanel(
                    "📄", 
                    Path.GetFileName(filePath)
                );
                DockPanel.SetDock(titlePanel, Dock.Top);
                mainPanel.Children.Add(titlePanel);
                
                // 创建预览内容
                var contentPanel = CreateContent(filePath);
                mainPanel.Children.Add(contentPanel);
                
                return mainPanel;
            }
            catch (Exception ex)
            {
                return PreviewHelper.CreateErrorPreview($"预览失败: {ex.Message}");
            }
        }
        
        private UIElement CreateContent(string filePath)
        {
            // 实现预览逻辑
            return new TextBlock { Text = "Hello World" };
        }
    }
}
```

## 常见问题

### Q1: 为什么某些格式无法预览？
**A:** 可能的原因：
- 文件损坏
- 缺少必要的编解码器或组件
- 文件大小超过限制
- 格式不在支持列表中

### Q2: 如何提高大文件预览速度？
**A:** 建议：
- 减小文件大小
- 使用SSD硬盘
- 增加系统内存
- 关闭其他占用资源的程序

### Q3: 可以添加自定义预览器吗？
**A:** 计划在 v2.0 引入插件系统，届时可以添加自定义预览器。

### Q4: 预览功能是否支持网络文件？
**A:** 目前仅支持本地文件，网络文件需先下载到本地。
