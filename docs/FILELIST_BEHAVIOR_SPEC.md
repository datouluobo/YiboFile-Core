# 文件列表操作行为规范

参考 Windows 资源管理器的标准行为，统一文件列表的交互逻辑。

## 1. 鼠标操作

### 1.1 左键单击
- **行为**：选择文件/文件夹
- **效果**：
  - 单选：清除之前选择，选中当前项
  - 显示文件预览、文件信息、备注
  - 如果是图片，执行 AI 标签预测
  - 如果是文件夹且大小未计算，立即计算大小

### 1.2 左键双击
- **文件夹**：进入文件夹（导航到该路径）
- **文件**：使用默认程序打开文件
- **列头/分隔线**：不触发文件操作
- **空白区域**：返回上一级目录

### 1.3 空白区域单击
- **行为**：清除文件列表选择
- **效果**：清除预览、文件信息、备注

### 1.4 中键点击
- **文件夹**：在新标签页打开文件夹
- **文件**：在新标签页打开文件所在文件夹

### 1.5 多选操作
- **Ctrl+左键**：切换单个项的选中状态（多选）
- **Shift+左键**：范围选择（从上次选择到当前项）
- **Ctrl+A**：全选所有项

### 1.6 拖拽操作
- **拖拽开始**：
  - 移动距离超过阈值（SystemParameters.MinimumHorizontalDragDistance）时启动
  - 保持选中项高亮显示
  - 显示拖拽视觉效果
- **拖拽到文件夹**：
  - 默认：移动（同驱动器）或复制（跨驱动器）
  - Ctrl：强制复制
  - Shift：强制移动
  - Ctrl+Shift：创建快捷方式
- **拖拽到库**：添加到库（Link 效果）
- **拖拽到快速访问**：添加到快速访问
- **拖拽到外部**：导出文件

## 2. 键盘操作

### 2.1 文件操作快捷键
- `Ctrl+A`：全选文件列表中的所有项
- `Ctrl+C`：复制选中的文件/文件夹
- `Ctrl+V`：粘贴到当前目录
- `Ctrl+X`：剪切选中的文件/文件夹
- `Delete`：删除选中的文件/文件夹（不在文本框中时）
- `F2`：重命名选中的文件/文件夹
- `F5`：刷新当前文件列表
- `Alt+Enter`：显示文件/文件夹属性

### 2.2 导航快捷键
- `Backspace` / `←`（列表视图）：返回上一级目录
- `Enter`：打开选中的文件/文件夹
- `方向键`：
  - `↑` / `↓`：上下移动选择
  - `←` / `→`：在缩略图视图中左右移动选择
  - `Home`：选择第一项
  - `End`：选择最后一项
- **缩略图视图**：方向键支持多列导航

## 3. 选择行为

### 3.1 单选
- 单击文件/文件夹项：选择该项，清除其他选择

### 3.2 多选
- **Ctrl+左键单击**：切换单个项的选中状态（如果已选中则取消，未选中则选中）
- **Shift+左键单击**：范围选择（从上次选择到当前项之间的所有项）
- **Ctrl+A**：全选所有项
- **注意**：ListView 默认支持这些多选操作，事件处理不应阻止默认行为

### 3.3 选择变化响应
- 更新文件预览
- 更新文件信息面板
- 更新备注内容
- 图片文件触发 AI 标签预测
- 文件夹触发大小计算（如果未计算）

## 4. 拖拽行为

### 4.1 拖拽源（文件列表）
- **启动条件**：鼠标移动距离超过阈值
- **选中状态保持**：拖拽过程中保持选中项高亮
- **视觉效果**：显示拖拽提示窗口，跟随鼠标移动

### 4.2 拖拽目标
- **文件夹**：移动/复制到目标文件夹
- **库**：添加到库（不移动文件）
- **快速访问**：添加到快速访问
- **外部程序**：导出文件

### 4.3 拖拽操作类型
- **默认**：根据源和目标位置决定（同驱动器=移动，跨驱动器=复制）
- **Ctrl**：强制复制
- **Shift**：强制移动
- **Ctrl+Shift**：创建快捷方式

## 5. 特殊处理

### 5.1 列头区域
- 点击列头：不触发文件选择，触发排序
- 双击列分隔线：自动适配列宽
- 拖拽列分隔线：调整列宽

### 5.2 空白区域
- 单击：清除选择
- 双击：返回上一级目录

### 5.3 库模式
- 双击文件夹：切换到路径模式并导航
- 拖拽到库：添加到库

### 5.4 标签模式
- 选择图片：触发 AI 标签预测
- 显示预测结果在标签编辑面板

## 6. 事件处理顺序

1. **PreviewMouseDoubleClick**：优先处理，拦截列头双击
2. **MouseDoubleClick**：备用处理双击
3. **PreviewMouseLeftButtonDown**：记录按下位置，处理列头点击
4. **PreviewMouseDown**：处理中键/Ctrl+左键
5. **MouseLeftButtonUp**：处理空白区域点击
6. **SelectionChanged**：响应选择变化
7. **PreviewKeyDown**：处理键盘快捷键

## 7. 实现要求

- 所有事件处理必须检查列头区域，避免干扰排序和列宽调整
- 拖拽操作必须保持选中状态高亮
- 选择变化必须及时更新预览和信息
- 键盘操作必须支持标准文件管理器快捷键
- 空白区域操作必须正确识别并处理
