# MainWindow 重构 - 第二阶段 (Phase 2)

## 目标描述
继续通过将不同功能区域提取为 partial classes（分部类）来降低 `MainWindow.xaml.cs` 的复杂度。本阶段重点关注菜单事件处理、搜索功能和预览逻辑。

## 计划变更

### MainWindow.xaml.cs (重构)
- 将菜单相关的事件处理程序移动到 `MainWindow.Menu.cs`。
- 将搜索相关的方法（不包括已经在 Tabs 中的）移动到 `MainWindow.Search.cs`。
- 将预览初始化和处理逻辑移动到 `MainWindow.Preview.cs`。

### [新建] MainWindow.Menu.cs
- 将包含 `Exit_Click`, `SelectAll_Click`, `ViewLargeIcons_Click` 等方法。
- 将包含委托给 `MenuEventHandler` 的逻辑。

### [新建] MainWindow.Search.cs
- 将包含各种 `Search*_Click` 处理程序。
- 将包含 `InitializeSearch`, `OnSearchTextChanged` 等方法。

### [新建] MainWindow.Preview.cs
- 将包含 `InitializePreview`, `ClosePreview` 以及预览相关的事件处理程序。

## 验证计划

### 自动化验证
- 运行 `dotnet build` 确保代码移动后没有编译错误。

### 手动验证
- **菜单**: 验证 "文件" -> "退出"、视图选项、设置菜单项是否正常工作。
- **搜索**: 验证在搜索框中输入能否过滤文件，以及搜索按钮是否工作。
- **预览**: 验证选中文件后，预览面板（如适用）是否显示预览。
