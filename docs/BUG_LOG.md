# YiboFile Bug Log

## [Unresolved] 副列表库加载失效
- **问题描述**: 在双面试图模式下，使用侧边导航栏切换副列表显示的“库”内容时，文件列表内容为空（首次进入不刷新）。
- **重现步骤**:
  1. 开启双面板模式。
  2. 激活副面板（右侧）。
  3. 在左侧 NavigationRail/Panel 中点击一个“库”（如视频、图片）。
  4. 观察到副面板路径条更新，但文件列表区域显示为空。
- **临时方案**: 切换一次标签页或重新导航可恢复刷新。
- **初步分析**: 侧边导航栏触发的消息可能只被主面板订阅，或者副面板在接收到 `LibraryChangedMessage` 时未能正确触发数据加载流。
- **记录时间**: 2026-02-07 14:30
- **版本**: v1.0.380
