# 更新日志

## [1.5.0] - 2025-01-20

### 新增功能 ✨

#### 缩略图视图与视图切换 🖼️
- **恢复缩略图视图**：重新实现了缩略图查看功能。
- **视图切换**：文件浏览器工具栏新增视图切换按钮（列表/缩略图）。
- **按需加载**：确保缩略图生成仅在缩略图视图下运行，大幅优化资源使用。

### 优化 ⚡

#### Office 文档预览优化 📄
- **异步预览**：.doc 文档预览转换现在完全在后台运行，防止界面卡死。
- **加载状态提示**：添加了“正在检测/转换...”的加载提示。
- **错误处理**：改进了缺少 Word 或转换失败时的用户提示。

#### 性能改进 🚀
- **列表视图优化**：切换到列表视图时强制停止所有缩略图生成任务。
- **SVG/PSD 预览**：图片预览加载改为异步，提升浏览流畅度。
- **文件夹大小计算**：添加任务取消逻辑，防止快速切换时计算任务堆积。

### 修复 🐛

#### 稳定性与 Bug 修复 🛠️
- **启动崩溃修复**：修复了 GroupedResultsViewer 导致的 System.ArgumentException。
- **信息面板**：修复了文件选中时信息面板和备注面板不更新的问题。
- **全局异常处理**：增强了应用程序的健壮性。

## [1.2.0] - 2025-01-15

### 新增功能 ✨

#### 统一标签页系统 🏷️
- **浏览器式标签页**：库和路径统一使用同一套标签页系统
- **标签页类型**：支持路径类型和库类型标签页
- **标签页关闭按钮**：
  - 悬停时显示关闭按钮，不改变标签宽度
  - 文本居中，悬停时平滑切换为关闭按钮
  - 关闭按钮颜色随标签状态变化

#### 文件预览增强 📄
- **.lnk 快捷方式预览**：显示快捷方式目标路径和文件信息
- **文件夹预览双击**：支持双击预览区文件/文件夹打开
- **文件夹预览打开**：快捷方式预览支持打开目标文件夹

#### 文件夹大小计算优化 📊
- **数据库缓存**：使用独立数据库文件缓存文件夹大小
- **智能计算**：
  - 文件列表刷新时立即显示缓存的大小
  - 选中文件夹时立即计算大小
  - 首次打开文件夹时计算所有子文件夹大小
- **递归计算**：文件夹大小包含所有子级文件夹
- **性能优化**：
  - 限制计算深度和文件数量
  - 利用CPU闲置时间进行后台计算
  - 自动清理不存在的文件夹缓存数据

#### UI/UX 改进 🎨
- **按钮分组**：标题栏按钮按功能分组，组间间距增大（16px）
- **库功能按钮整合**：将列1的库功能按钮移动到标题栏
- **对话框统一**：删除库对话框使用统一的ConfirmDialog样式
- **分隔符对齐**：标题栏分隔符与列1/列2分割器对齐

#### 性能优化 ⚡
- **鼠标操作优化**：减少不必要的布局更新和命中测试
- **文件夹大小计算**：异步计算，不阻塞UI
- **程序关闭速度**：优化关闭流程，移除不必要的等待

### 修复 🐛
- 修复标签页高度变化问题
- 修复标签页关闭按钮显示问题
- 修复按钮消失问题

## [1.1.0] - 2025-01-02

### 新增功能 ✨

#### 库功能 📚
- Windows 风格的库功能，可以聚合多个文件夹位置到一个虚拟库中
- 支持一个库包含多个不同路径的文件夹
- 库管理窗口：创建、重命名、删除库，添加/删除库位置
- 库右键菜单：重命名、删除库、管理位置、在资源管理器中打开、刷新列表
- 库模式下显示所有位置的合并文件列表
- 支持从文件列表添加文件/文件夹到库
- 数据库迁移：自动从单路径库迁移到多路径库结构

#### 收藏功能 ⭐
- 收藏常用文件和文件夹，快速访问
- 文件夹和文件分类显示（文件夹在上，文件在下）
- 拖拽排序：支持拖拽调整收藏项顺序，顺序持久化保存
- 同名区分：自动添加父文件夹名称区分同名文件/文件夹
- 整行背景显示：列表项有明显的背景色和分隔线
- 鼠标悬停和选中状态高亮显示
- 双击打开收藏的文件夹或文件
- 右键菜单删除收藏

#### UI/UX 改进 🎨
- **统一显示效果**：
  - 驱动器、快速访问、收藏统一使用 ListBox 控件
  - 统一的背景色（#EEEEEE）和分隔线
  - 统一的鼠标悬停（#E3F2FD）和选中（#BBDEFB）效果
- **列头设置持久化**：
  - 列宽度自动保存和恢复
  - 列顺序支持调整并持久化保存
  - 按 Tag 标识列，支持任意顺序
- **交互优化**：
  - 双击文件列表空白区域返回上一级目录
  - 所有对话框支持 Enter 确认和 Esc 取消
  - 驱动器列表高度自适应（移除最大高度限制）
  - 文件夹双击响应优化，不再等待计算大小

#### 性能优化 ⚡
- 文件夹大小计算完全异步，不影响双击响应
- 列宽度保存使用延迟机制（1秒），避免频繁写入

### 修复 🐛
- 修复库列表不刷新的问题
- 修复选择库后文件列表不更新的问题
- 修复库模式下文件夹无法双击进入的问题
- 修复库没有位置时的提示显示方式（改为页面内显示而非弹窗）
- 修复收藏成功提示框（改为静默完成）

## [1.0.0] - 2025-11-02

### 新增功能 ✨

#### 核心功能
- 三栏式文件管理器布局（导航栏、文件列表、预览/备注区）
- 完整的文件操作功能（新建、复制、粘贴、剪切、删除、重命名）
- Windows 标准快捷键支持（Ctrl+C/V/X、Delete、F2、F5、Alt+Enter 等）
- 智能键盘导航（方向键、Home/End、Enter、Backspace）
- 文件系统实时监控和自动刷新（300ms 防抖）

#### 预览功能 🎨
- **文本文件**：支持多种编码（UTF-8、GBK、GB2312 等）
- **图片文件**：PNG、JPG、GIF、BMP、SVG、ICO、TIFF、WebP
- **视频文件**：MP4、AVI、MKV、WMV、MOV、FLV（带播放控制）
- **音频文件**：MP3、WAV、FLAC、AAC、OGG、WMA
- **文档文件**：
  - DOCX/DOC：HTML 转换预览 + 转换为 DOCX 功能
  - PDF：WebView2 渲染
  - XLSX：表格数据预览（支持多工作表）
- **压缩包**：ZIP、RAR、7Z 文件列表预览（支持中文编码自动检测）
- **文件夹**：显示文件夹内容列表（与左侧列表一致的界面）

#### 新建文件功能 📝
- **Office 文档**：Word、Excel、PowerPoint（使用 COM 自动化创建实际格式）
- **图片文件**：
  - PNG、JPG、GIF、BMP：500x500 像素空白图片
  - SVG：矢量图模板
- **代码文件**：C#、Python、Java、JavaScript、HTML、CSS（带模板代码）
- **配置文件**：JSON、XML、INI、Markdown
- **脚本文件**：BAT、PowerShell
- 支持自定义扩展名
- 重名自动添加序号

#### UI/UX 改进 🎯
- **创建时间列**：
  - 简洁格式显示（s/m/h/d/mo/y）
  - 彩虹色渐变着色：
    - 秒（s）- 绿色 + 粗体
    - 分钟（m）- 青色 + 半粗体
    - 小时（h）- 蓝色
    - 天（d）- 紫色
    - 月（mo）- 棕色
    - 年（y）- 深灰色
- **文件夹预览**：右侧预览区显示与左侧一致的文件列表界面
- **统一对话框风格**：
  - 现代化设计（圆角、无边框、阴影）
  - 支持 Enter/Esc 快捷键
  - 一致的视觉风格
- **布局优化**：
  - 列3与列2水平对齐
  - 响应式布局
  - 可调整列宽

#### 标签和备注 🏷️
- 为文件和文件夹添加自定义标签
- 添加和编辑备注信息
- 按标签筛选文件
- 搜索功能

### 技术实现 🔧

#### 依赖包
- Microsoft.Web.WebView2 (1.0.3537.50) - PDF 和 DOCX 预览
- ExcelDataReader (3.7.0) - Excel 文件读取
- ExcelDataReader.DataSet (3.7.0) - Excel 数据集支持
- System.Text.Encoding.CodePages (9.0.10) - 扩展编码支持

#### 架构设计
- **模块化预览系统**：
  - `IPreviewProvider` 接口
  - `PreviewFactory` 工厂模式
  - 按文件类型分离的预览类
- **文件类型管理**：`FileTypeManager` 统一管理文件类型信息
- **配置管理**：`ConfigManager` 处理应用配置
- **数据库管理**：`DatabaseManager` 管理标签和备注数据

#### 代码组织
```
Previews/
├── IPreviewProvider.cs      # 预览提供者接口
├── PreviewFactory.cs        # 预览工厂
├── PreviewHelper.cs         # 预览辅助工具
├── TextPreview.cs           # 文本预览
├── ImagePreview.cs          # 图片预览
├── VideoPreview.cs          # 视频预览
├── AudioPreview.cs          # 音频预览
├── DocumentPreview.cs       # 文档预览
├── ArchivePreview.cs        # 压缩包预览
├── FolderPreview.cs         # 文件夹预览
└── DocxToHtmlConverter.cs   # DOCX 转 HTML
```
