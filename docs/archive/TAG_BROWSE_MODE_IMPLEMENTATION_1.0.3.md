## 使用说明

1. **切换到标签页面**：点击标题栏的"标签"按钮
2. **切换浏览/编辑模式**：点击列1底部右侧的浏览/编辑图标按钮
3. **打开分组管理**：点击列1底部右侧的分组管理按钮
4. **浏览模式**：点击标签查看该标签的文件列表
5. **编辑模式**：使用完整的TagTrain训练功能
